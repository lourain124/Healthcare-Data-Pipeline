# 🧠 Project Healthcare Data Pipeline

## 📌Description
End-to-end data engineering project using the UCI Thyroid Disease dataset. Implements an ETL pipeline with data ingestion, cleaning, and transformation into a PostgreSQL data warehouse (star schema). Includes data quality checks and dashboard visualization with Grafana for healthcare analytics.

## ⚙️ Tecnologies
- Python
- Pandas
- PostgreSQL
